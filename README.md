โปรเจกต์ "EmotiFace LED" นี้มีวัตถุประสงค์เพื่อสร้างระบบแสดงสีหน้าทางดิจิทัลที่สามารถใช้ในงานคอสเพลย์ได้ 
โดยใช้แผง LED ขนาด 32x8 ที่สามารถปรับเปลี่ยนเป็นสีหน้าต่าง ๆ ผ่านการควบคุมทางไกลหรือเซ็นเซอร์ในอนาคต 
เป้าหมายหลักคือเพื่อให้คอสเพลเยอร์สามารถแสดงอารมณ์ผ่านหน้ากากหรือหน้าจอ LED ที่แสดงสีหน้าแบบต่าง ๆ ได้ 
เพิ่มความสนุกสนานและเพิ่มระดับความสมจริงของตัวละคร
