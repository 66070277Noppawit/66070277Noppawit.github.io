EmotiFace LED Project
Source Code Team Member

Abstract

โครงการนี้มีวัตถุประสงค์ในการพัฒนา “EmotiFace LED” ซึ่งเป็นอุปกรณ์ที่สามารถแสดงสีหน้าและอารมณ์ผ่านแผงไฟ LED ชนิด WS2812B ขนาด 32x8 เพื่อนำไปใช้ในกิจกรรม Cosplay โดยตัวอุปกรณ์นี้จะสามารถแสดงสีหน้าในลักษณะต่างๆ เช่น การยิ้ม ความเศร้า และการแสดงออกทางอารมณ์อื่นๆ ได้อย่างสมจริง และมีความยืดหยุ่นในการใช้งานผ่านการควบคุมจากระยะไกลด้วยระบบ IoT (Internet of Things) อุปกรณ์ “EmotiFace LED” นี้ถูกออกแบบมาเพื่อให้สามารถปรับเปลี่ยนสีหน้าได้ตามต้องการผ่านคำสั่งจากผู้ใช้ ทำให้การแสดงสีหน้าสามารถปรับให้สอดคล้องกับสถานการณ์ต่างๆ ได้ โดยใช้ระบบไฟ LED ซึ่งทำให้เกิดความชัดเจนและสามารถดึงดูดความสนใจได้ดี นอกจากนี้ อุปกรณ์นี้ยังมีขนาดกะทัดรัดและง่ายต่อการติดตั้ง โดยเฉพาะสำหรับนักแสดง Cosplay ที่ต้องการการแสดงออกทางสีหน้าที่โดดเด่นและน่าดึงดูด
ระบบนี้ยังสามารถเชื่อมต่อกับแพลตฟอร์มต่างๆ ผ่านระบบ MQTT Protocol ทำให้สามารถควบคุมอุปกรณ์ได้จากระยะไกล ซึ่งช่วยเพิ่มความสะดวกในการใช้งาน และเปิดโอกาสให้นำไปใช้ในกิจกรรมต่างๆ ที่ต้องการการสื่อสารด้วยภาพและแสงสี นอกจากนี้ ยังสามารถพัฒนาและปรับแต่งเพิ่มเติมเพื่อใช้งานในด้านอื่นๆ เช่น การแสดงสีหน้าในสื่อบันเทิง งานนิทรรศการ หรืองานสื่อสารเชิงโต้ตอบ

ระบบนี้ยังสามารถเชื่อมต่อกับแพลตฟอร์มต่างๆ ผ่านระบบ MQTT Protocol ทำให้สามารถควบคุมอุปกรณ์ได้จากระยะไกล ซึ่งช่วยเพิ่มความสะดวกในการใช้งาน และเปิดโอกาสให้นำไปใช้ในกิจกรรมต่างๆ ที่ต้องการการสื่อสารด้วยภาพและแสงสี นอกจากนี้ ยังสามารถพัฒนาและปรับแต่งเพิ่มเติมเพื่อใช้งานในด้านอื่นๆ เช่น การแสดงสีหน้าในสื่อบันเทิง งานนิทรรศการ หรืองานสื่อสารเชิงโต้ตอบ

Project Details

การทำงานของระบบ แผง LED ขนาด 32x8 จะทำหน้าที่แสดงสีหน้าต่าง ๆ ตามคำสั่งที่ส่งมาจากอุปกรณ์ควบคุม เช่น มือถือ หรือสั่งงานผ่านแอปพลิเคชันที่เชื่อมต่อ Wi-Fi โดยการควบคุมทำผ่านซอฟต์แวร์บน Arduino ใช้ไลบรารี FastLED และการส่งข้อมูลผ่านระบบ MQTT เพื่อความสะดวกและความรวดเร็วในการแสดงผลแบบทันทีทันใด

Why our project??

  • เปลี่ยนการแสดงผลได้หลากหลายรูปแบบ: สามารถเปลี่ยนสีหน้าได้ตามต้องการเพื่อแสดงอารมณ์หลากหลาย เช่น ยิ้ม โกรธ หรือเศร้า
  • ควบคุมได้แบบไร้สาย: ผู้ใช้งานสามารถสั่งการและปรับเปลี่ยนการแสดงผลได้จากระยะไกลผ่าน Wi-Fi สะดวกต่อการใช้งานขณะ Cosplay
  • การแสดงสีได้หลากหลาย: รองรับการแสดงสีได้มากถึง 16,777,216 สี สร้างความสมจริงให้กับตัวละครและเพิ่มความมีชีวิตชีวา

Benefit of Our Project

  • ประยุกต์ใช้ในการแสดงหรือคอนเสิร์ต: แผงไฟ LED สามารถใช้เป็นอุปกรณ์เสริมสำหรับการแสดงบนเวที เพิ่มความโดดเด่นให้กับผู้สวมใส่
  • สร้างสรรค์ในการ Cosplay: เหมาะสำหรับการใช้งาน Cosplay ที่ต้องการแสดงอารมณ์หรือคาแรคเตอร์ของตัวละครได้สมจริงยิ่งขึ้น
  • ใช้งานในเชิงการศึกษาและพัฒนาทักษะ: โปรเจกต์นี้สามารถใช้เป็นพื้นฐานในการเรียนรู้การเขียนโปรแกรมควบคุมฮาร์ดแวร์และระบบ IoT รวมถึงการทำงานของระบบสื่อสารแบบไร้สาย

Arduino UNO R4 Wifi
